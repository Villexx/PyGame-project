# 🏎️ Counter Flow

## Objetivo
Jogo 2D vertical de desvio de tráfego em alta velocidade (corrida infinita) com sistema de pontuação progressiva.

## Motivação
Projeto acadêmico desenvolvido para a disciplina de Linguagem de Programação.

## Tecnologias Utilizadas
* Python 3.x
* PyGame >= 2.6.1

## Instalação e Configuração
Abra o terminal na pasta raiz do projeto dentro do VSCode e escolha uma das abordagens abaixo para instalar as dependências listadas no arquivo `requirements.txt`:

* **Opção 1 (Instalação Isolada - Recomendado):** Cria um ambiente virtual (`venv`) para isolar o PyGame dentro da pasta do projeto.
  * **Criar o ambiente virtual:** `python -m venv venv`
  * **Ativar no Windows (PowerShell):** `.\venv\Scripts\Activate.ps1`
  * **Ativar no Mac/Linux (Terminal):** `source venv/bin/activate`
  * **Instalar as dependências:** `pip install -r requirements.txt`
* **Opção 2 (Instalação Global):** Instala o PyGame diretamente no escopo global do seu usuário.
  * **Instalar as dependências:** `pip install -r requirements.txt`

## Como Executar
1. Abra a pasta do projeto no VS Code.
2. Certifique-se de ter concluído a instalação das dependências.
3. Execute o arquivo principal a partir do terminal para iniciar o jogo:
   `python main.py`
